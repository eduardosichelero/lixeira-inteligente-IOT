## Instalação

1. Instale as dependências:

   ```bash
   pip install pyserial
   ```

2. Execute o client:

   ```bash
   python client/client.py
   ```

3. Execute o server para monitorar eventos e passar os comandos:

   ```bash
   python server/server.py
   ```
